## 🧑‍💻 About Me

I'm **Md Khokanuzzaman Khokan** — a **Senior Software Engineer** at [Square Health Ltd](https://squarehealth.com.bd), building production-grade mobile applications that serve thousands of users daily.

Beyond my day job, I'm deeply invested in the **AI-powered developer tooling** space — exploring agentic workflows, vibe coding, and how AI is reshaping software engineering from the ground up.

I also run **[Troubleshoot Bangla](https://www.youtube.com/@TroubleshootBangla)** — a growing Bangla-language tech education brand where I break down Flutter, React Native, and AI tools for the next generation of developers.

```
🔭  Currently building    →  PocketPilot AI — a Bengali-first personal finance app (Flutter + AI)
🧠  Deep diving into      →  Agentic AI, Claude Code, Cursor, n8n Automation
🎓  Teaching via           →  Troubleshoot Bangla (YouTube + Facebook)
🌐  Portfolio & Courses    →  khokan.me
⚡  Fun fact              →  I mass-produce content — 1 recording session = 5+ pieces across platforms
```
